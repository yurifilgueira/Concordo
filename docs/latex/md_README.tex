T\+U\+R\+MA\+: 02\hypertarget{md_README_autotoc_md3}{}\doxysection{Como executar o Concordo}\label{md_README_autotoc_md3}
Certifique-\/se de ter o g++ e o C\+Make instalados em seu sistema antes de prosseguir.


\begin{DoxyEnumerate}
\item No terminal, execute o comando {\ttfamily cmake .} para gerar os arquivos de configuração do C\+Make.

1.\+1. Caso a sua versão seja diferente da do arquivo {\ttfamily C\+Make\+Lists.\+txt}, presente nesse projeto, não tem problema. Basta acessar o arquivo citado, e alterar a versão que está no arquivo para a versão que está no seu dispositivo.
\item Em seguida, execute o comando {\ttfamily make} para compilar o programa.
\item Após a compilação bem-\/sucedida, execute o programa digitando {\ttfamily ./program} no terminal.

3.\+1. Caso você possua um script — um arquivo {\ttfamily .txt} — com os comandos do Concordo, basta inserir {\ttfamily ./program nome\+Do\+Arquivo.\+txt} no terminal.

3.\+2. Caso o script não contenha o comando {\ttfamily quit} — comando que será explicado em breve — o Concordo continurá rodando normalmente, e você poderá inserir os comandos que desejar.

3.\+3. Neste projeto, você encontrará 2 scripts, o {\ttfamily script\+Valid.\+txt} — que executa os comandos de maneira válida — e o {\ttfamily script\+Invalid.\+txt} — que executa os comandos de maneira errada, e faz com que o programa mostre as mensagens de erro para o usuário, sem crash no programa —, basta seguir o passo 3.\+1 para executá-\/los. Só um por vez.

3.\+4. Os comandos — executados de maneira correta e errada — das funcionalidades relacionadas aos canais, se encontram no {\ttfamily script\+Valid.\+txt}.
\end{DoxyEnumerate}\hypertarget{md_README_autotoc_md4}{}\doxysection{Guia de uso}\label{md_README_autotoc_md4}
O {\ttfamily Concordo} funciona através de comandos inseridos no terminal. Portanto, aqui estão os comandos possíveis de se fazer e como fazer.

Ao iniciar o {\ttfamily Concordo} — sem estar logado —, você só poderá executar os três primeiros comandos. Caso tente executar algum comando além desses três — sem estar logado —, o programa acusará um erro.

Segue abaixo a lista de comandos possíveis\+:


\begin{DoxyEnumerate}
\item {\bfseries{create-\/user}}\+: Cria um usuário. Veja o exemplo\+:

{\ttfamily create-\/user $<$email$>$ $<$senha\+\_\+sem\+\_\+espaços$>$ $<$nome com espaços$>$}

{\ttfamily create-\/user ryangosling@example.\+com 12ab34cd Ryan Gosling}
\item {\bfseries{login}}\+: Realiza o login no Concordo. Veja o exemplo\+:

{\ttfamily login $<$nome$>$ $<$senha$>$}

{\ttfamily login ryangosling@example.\+com 12ab34cd}
\item {\bfseries{quit}}\+: Encerra o programa, pode ser usado a qualquer momento. Veja o exemplo\+:

{\ttfamily quit}
\item {\bfseries{disconnect}}\+: Desconecta o usuário que está atualmente logado. Veja o exemplo\+:

{\ttfamily disconnect}
\item {\bfseries{create-\/server}}\+: Cria um servidor. Veja o exemplo\+:

{\ttfamily create-\/server $<$nome$>$}

{\ttfamily create-\/server Javeiros-\/com-\/amor}
\item {\bfseries{set-\/server-\/desc}}\+: Permite definir ou alterar uma descrição para o servidor. Caso você não seja o dono do servidor, não poderá alterar ou definir a descrição do servidor. Veja o exemplo\+:

{\ttfamily set-\/server-\/desc $<$nome-\/do-\/servidor$>$ $<$descrição$>$}

{\ttfamily set-\/server-\/desc Javeiros-\/com-\/amor Seja bem-\/vindo ao servidor da melhor linguagem do mundo.}
\item {\bfseries{set-\/server-\/invite-\/code}}\+: Permite definir ou alterar um código de convite para o servidor. Somente pessoas com acesso a este código poderão entrar no servidor. Caso você não seja o dono do servidor, não poderá alterar ou definir o código de convite do servidor. Veja o exemplo\+:

{\ttfamily set-\/server-\/invite-\/code $<$nome-\/do-\/servidor$>$ $<$código-\/de-\/convite$>$}

{\ttfamily set-\/server-\/invite-\/code Javeiros-\/com-\/amor j1a2v3a4}
\item {\bfseries{list-\/servers}}\+: Lista os servidores presentes no Concordo. Veja o exemplo\+:

{\ttfamily list-\/servers}
\item {\bfseries{remove-\/server}}\+: Ao usar este comando, você, caso seja o dono, deletará o servidor. Veja o exemplo\+:

{\ttfamily remove-\/server $<$nome-\/do-\/servidor$>$}

{\ttfamily remove-\/server Javeiros-\/com-\/amor}
\item {\bfseries{enter-\/server}}\+: Permite entrar em um servidor. Entretanto, há dois tipos de servidores, os que possuem código de convite e os que não possuem. Veja o exemplo\+:

{\bfseries{Com código de convite\+:}}

{\ttfamily enter-\/server $<$nome$>$ $<$código-\/de-\/convite$>$}

{\ttfamily enter-\/server Javeiros-\/com-\/amor j1a2v3a4}

{\bfseries{Sem código de convite\+:}}

{\ttfamily enter-\/server $<$nome$>$}

{\ttfamily enter-\/server Javeiros-\/com-\/amor}

{\itshape É importante ressaltar que, ao entrar em um servidor, você entrará para a lista de participantes e ao mesmo tempo, estará visualizando o servidor.}
\end{DoxyEnumerate}
